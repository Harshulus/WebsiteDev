JavaScript Objects

An object is a collection of properties, and a property is an association between a name (or key) and a value. JavaScript objects are same as any other programming language. Example of JavaScript objects: A basketball is an object, with different properties. Different basketballs have different colors, designs/patterns, weight, are made of different material it is made of, etc. The same way, JavaScript objects can have properties, which define their characteristics of the respective object.

Example JavaScript Objects:

  • All basketball have the same properties, but the property values differ from basketball to basketball.
  • All basketball have the same methods, but the methods are performed at different times.

    JavaScript Objects "this" keyword

  • "this" keyword refers to the "owner" of the function.
  • "this" keyword refers to the global object.
  • Methods like call(), and apply() can refer this to any object.
