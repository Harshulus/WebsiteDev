HTML Web Forms


Forms are used to allow for user inputs. There are different types of forms such as: textbox, checkbox and radio buttons.

These forms can be created with the tag <form> and the contents inside the forms can be changed with the tags <input type="text">, <input type="checkbox">, <input type="radio"> and <input type="radio">.

Textbox

This allows for users to type inside the field.

<form action="/handler.php">
    <label for="box">Textbox:</label>
    <br>
    <input type="text" id="box1" name="box1>
    <br>
    <input type="submit" value="Submit">
</form>




Checkbox

This allows for users to select multiple options.

<form action="/handler.php">
    <input type="checkbox" id="option1" name="option1" value="option1">
    <label for="option1">Option 1</label>
    <br>
    <input type="checkbox" id="option2" name="option2" value="option2">
    <label for="option2">Option 2</label>
    <br>
    <input type="checkbox" id="option3" name="option3" value="option3">
    <label for="option3">Option 3</label>
    <br>
    <input type="submit" value="Submit">
</form>




Radio Buttons

This allows for users to select only one option.

<form action="/handler.php">
    <input type="radio" id="option1" name="option" value="option1">
    <label for="option1">Option 1</label>
    <br>
    <input type="radio" id="option2" name="option" value="option2">
    <label for="option2">Option 2</label>
    <br>
    <input type="radio" id="option3" name="option" value="option3">
    <label for="option3">Option 3</label>
    <br>
    <input type="submit" value="Submit">
</form>
