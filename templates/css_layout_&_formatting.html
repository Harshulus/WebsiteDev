CSS Layout & Formatting

CSS Layout

The structure of a website is commonly divided into a header, navigation bar, body and footer. The appearances of these sections can be formatted with CSS.


The header is located at the top of the page and mainly contains a logo.


The navigation bar contains the buttons which redirect to other pages.


The body is where all the main content goes, all the information that you want to provide to the user.


The footer is the bottom of the page and mainly contains the contact information and copyright.



CSS Formatting

Formatting HTML elements is easily done through changing the properties. Some of the commonly used formatting properties are:

  • font-size

    • Changes the size of the text

    p {
    font-size: 20px;
}


  • color

    • Changes the colour of the text

    p {
    color: white;
}


  • text-align

    • Changes the horizontal alignment of the text

    p {
    text-align: right;
}


  • background-color

    • Changes the background colour of the page

    body {
    background-color: blue;
}


  • margin

    • Changes the space around an element

    p {
    margin: 20px;
}


  • width

    • Changes the width of an element

    img {
    width: 125px;
}


  • height

    • Changes the height of an element

    img {
    height:50;
}
