Basics of CSS


What is CSS?

CSS stands for Cascading Style Sheets. CSS is a language which defines the aesthetics of HTML elements. Colouring, page margins, fonts, font sizes, page layout and other attributes are all formatted and customised with CSS.


How does CSS work?

CSS is mainly done in another file and is linked with the HTML file using '<link ref="stylesheet" href="file.css">'.

Example of CSS:

body {
    background-color: #222;
}

p {
    color: rgb(0, 0, 0);
}

CSS Structure


Selector

This is the HTML element which you want to format.

Declaration

This is the element's feature which you want to change.

Property

This is the variable name of the element's feature.

Value

This is the value which you are changing the feature to.
